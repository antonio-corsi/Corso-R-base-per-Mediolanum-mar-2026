Questo repo contiene tutto il materiale didattico del corso.

Si può scaricare tutto premendo il bottone verde "Code" e poi "Donwload ZIP".

Si può scaricare il singolo file facendo click sopra di esso e poi in alto a dx premendo il bottone "Download raw file" (la freccia).

I file del repo sono apribili e editabili ma non eseguibili. Si consiglia di NON editarli perchè possono essere successivamentye sovrascritti da github. 
Github è un ambiente di transito, non di lavoro.

L'attualità o meno del singolo file può essere desunta dalla etichetta o dalla data.
